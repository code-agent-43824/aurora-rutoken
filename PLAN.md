# План проекта aurora-rutoken

Правила работы с планом — см. [CLAUDE.md](CLAUDE.md). Кратко:

1. Любое действие **сначала** фиксируется здесь как задача (`[ ]`).
2. Затем выполняется.
3. Затем отмечается `[x]` с датой, а в [docs/JOURNAL.md](docs/JOURNAL.md) добавляется запись (что, почему, коммиты).

Статусы: `[ ]` — запланировано; `[~]` — в работе; `[x]` — выполнено; `[!]` — заблокировано (причина рядом с задачей).
