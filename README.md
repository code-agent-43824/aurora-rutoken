# aurora-rutoken

Тестовое приложение для ОС Аврора, работающее с токенами **Рутокен ЭЦП 3.0** по **USB** и **NFC**.

Стек: нативный Qt/C++ + QML (Аврора SDK, RPM-пакет). Доступ к токену: PKCS#11 (`librtpkcs11ecp.so`) поверх PC/SC (`pcscd`): USB — через CCID-handler, NFC — через NFC-стек ОС (`nfcd`).
